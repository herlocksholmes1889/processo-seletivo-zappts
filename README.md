## Resolução de case para o processo de estágio da Zappts

Código criado após uma dinâmica em grupo, com o fim de entrar para o programa de estágio em backend da consultoria Zappts. É um projeto simples que visa detectar viés de gênero em descrições de vagas para desenvolvimento de software, e oferecer sugestões de como evitá-lo.

Esta é uma atualização simples que será committada ao repositório principal no decorrer da demonstração de Protocolo HTTP e Autenticação.
